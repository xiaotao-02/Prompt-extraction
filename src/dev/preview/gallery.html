先在本仓库根目录执行 npm run dev（默认 http://localhost:5173）。 以下为真实 React / 真实 STYLE + panelHtml，随源码与 HMR 更新；与 chrome-extension:// 里仍会存在行为差异（预览使用内存 chrome.storage 与占位 runtime.sendMessage）。顶部标签切换预览；链接可加 #popup、#options、#panel；快捷键 1 / 2 / 3（本页焦点时）。

占位库条目（同源 IndexedDB，见 src/dev/preview/seedPreviewLibrary.ts）： PREVIEW_DEMO_PRIMARY_ID · preview-demo-1（双版本）、 PREVIEW_DEMO_SD_TAGS_ID · preview-demo-2、 PREVIEW_DEMO_REFINED_PAIR_ID · preview-demo-3、 PREVIEW_DEMO_TEN_VERSIONS_ID · preview-demo-10v（10 条版本链；Popup 上徽章为 v10 表示共 10 个版本）。 若本地库非空，预览页每次启动仍会补写缺失的上述占位 id（无需为看 10 版本而清空整库）。 浮动面板预览 iframe 默认加载 preview-demo-10v。本聚合页内的 Panel iframe URL 不带 hash， 需在单独 Tab 打开 /src/dev/preview/panel-preview.html#focus=preview-demo-1 （或 item= 别名）可切换占位记录。
