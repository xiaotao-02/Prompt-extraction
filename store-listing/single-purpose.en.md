# Single Purpose (English)

> Maps to form field: Privacy practices → Single purpose

This extension does exactly one thing:

**When the user right-clicks an image, animation or video on a web page, the extension sends that media (or the first frame of the animation / the current frame of the video) to the vision LLM provider that the user has personally selected in Options. The provider returns a prompt suitable for AI image generation, which is then displayed in a floating panel at the bottom-right of the page so the user can copy it.**

All ancillary features — local history, version editing, conversational refine, settings sync, manual update check — directly serve this single purpose. The extension contains no capabilities unrelated to "image → prompt".
