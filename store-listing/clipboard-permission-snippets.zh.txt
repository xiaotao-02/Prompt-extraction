# 剪贴板权限 — 开发者后台「隐私权规范 / Permission justifications」可粘贴文案
# 与代码一致：SetupGuide 从剪贴板导入 JSON；复制操作使用 clipboardWrite。
# 若平台字数极短，用「短版」；否则用「完整版」。

## clipboardRead（完整版）

仅在用户于扩展「设置 / 选项页」中主动点击「从剪贴板粘贴」类操作时，读取系统剪贴板中的纯文本，用于将用户自行备份或迁移的 JSON 格式 API 配置填入导入框并完成本地配置导入。不会后台静默读取剪贴板，也不将剪贴板内容用于广告或与该功能无关的目的。

## clipboardRead（短版）

仅在用户点击导入时读取剪贴板中的 JSON 文本，用于迁移/导入 API 配置。

## clipboardWrite（完整版，若平台另要求说明）

仅用于用户明确点击「复制」时：结果面板、工具栏弹窗，以及设置/提示词库中的复制按钮，将生成或选中的提示词或配置写入剪贴板。不会在无用户点击的情况下静默写入。

## clipboardWrite（短版）

仅在用户点击「复制」时把提示词或配置写入剪贴板。
