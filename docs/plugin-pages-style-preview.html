插件 UI 预览（与源码同步）

旧的纯 CSS 静态示意已改为「在 Vite dev 下运行真实组件」；改 popup / options / content/panel 后， 保存并在浏览器里刷新聚合页即可看到 HMR 效果。

  1. 在项目根目录运行 npm run dev（端口见终端，默认 5173）；就绪后会自动打开聚合预览页。若不想自动弹窗，设置环境变量 DEV_PREVIEW_NO_OPEN=1（PowerShell：$env:DEV_PREVIEW_NO_OPEN='1'; npm run dev）。
  2. 聚合地址（也可手动打开）： http://localhost:5173/__dev__/ui-preview （可加 #popup / #options / #panel / #contextmenu 选中对应预览；聚合页快捷键 1–4）
